南京邮电大学材料考研信息采集与分析系统

本项目面向南京邮电大学材料科学与工程相关考研公开信息，整理招生简章、专业目录、考试大纲、复试细则、拟录取名单、调剂信息等内容，并通过静态网页进行展示与统计分析。

当前版本 V1 静态展示版

数据由本地 Python 爬虫生成，网页可部署到 GitHub Pages。

系统模块

  • 本地爬虫：低频采集公开页面，生成 JSON 与 CSV。
  • 数据列表：支持类别、来源和关键词检索。
  • 分析图表：展示类别、来源和年份分布。
  • 静态部署：无需后端框架和数据库。

采集边界

  • 只采集公开网页信息。
  • 不采集登录、验证码、付费墙后的内容。
  • 不采集身份证号、手机号等个人隐私信息。
  • 遇到 403、429 或验证码页面会跳过。
